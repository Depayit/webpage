Depayit คืออะไร?

เราคือเป็นแพลตฟอร์มตัวกลาง (P2P Soft Escrow) คอยกำกับดูแลธุรกรรมให้ระหว่างผู้ซื้อและผู้ขาย ตามประมวลกฏหมายแพ่งและพาณิชย์ เราไม่ใช่ธนาคารและไม่เตะต้องเงินค่าสินค้าของผู้ใช้โดยตรงเพราะเรามีระบบ (Auto Release) การสั่งปล่อยเงินอัตโนมัติเมื่อครบเงื่อนไขตามที่ได้กำหนดไว้ และยังให้ความสำคัญมากๆในเรื่องความโปร่งใสและเป็นกลางดำเนินการในรูปแบบ"Agency-Agreement" เพื่อให้คู่ค้าโอนเงินและส่งมอบสินค้า ได้โดยมีตัวกลางทางเทคนิคที่จัดการสถานะและเงื่อนไขการปล่อยเงิน ใช้งานง่าย ปลอดภัย โปร่งใส ซื้อขายสะดวกสบาย

Anywhere, Anytime

ใช้งานผ่าน Web Application บนเบราว์เซอร์ ไม่ต้องโหลดแอป ไม่ต้องสมัครสมาชิกให้ยุ่งยาก ใช้ได้กับทุก Social Media

Dispute Protection

มีบริการเปิดข้อพิพาท (Dispute) เพื่อความเป็นธรรม หากสินค้ามีปัญหา เราพร้อมดูแลและตรวจสอบหลักฐานให้ทุกเคส

ใช้งานง่ายแค่ 4 ขั้นตอน

ออกแบบมาเพื่อการซื้อขายบน Social Media โดยเฉพาะ

1. สร้างรายการ

ผู้ขายหรือผู้ซื้อสร้างลิงก์ ระบุรายละเอียดสินค้าและราคา

2. ชำระเงิน

ผู้ซื้อโอนเงิน Scan QRCode เงินถูกพักไว้ที่บัญชีกลาง (Cashflow)

3. ส่งสินค้า

ผู้ขายส่งของและแจ้งเลข Tracking ในระบบเพื่อยืนยันและแจ้งบัญชีรับเงิน

4. รับของ-รับเงิน

เมื่อผู้ซื้อได้รับของ ระบบจะโอนเงินอัตโนมัติให้ผู้ขายทันที

ผู้ขายจะได้รับเงินตอนไหน?

ผู้ขายจะได้รับเงินทันทีที่ผู้ซื้อกดยืนยัน "ได้รับสินค้าแล้ว" หรือเมื่อครบกำหนดระยะเวลาตรวจสอบสินค้า (Auto-release) หากผู้ซื้อไม่แจ้งปัญหาเข้ามา

ถ้าสินค้ามีปัญหา หรือผู้ขายโกง ทำอย่างไร?

ผู้ซื้อสามารถกดปุ่ม "แจ้งปัญหา/ข้อพิพาท" (Dispute) ระบบจะระงับการโอนเงินทันที และ Depayit จะเข้ามาเป็นคนกลางตรวจสอบหลักฐาน

ต้องดาวน์โหลดแอปหรือไม่?

ไม่ต้องดาวน์โหลดครับ Depayit เป็น Web Application สามารถใช้งานผ่านเบราว์เซอร์ (Chrome, Safari) บนมือถือหรือคอมพิวเตอร์ได้ทันที
