ข้อกำหนดและเงื่อนไขการใช้บริการ

อัปเดตล่าสุด: 25 มกราคม 2026

i บทนำ

ข้อตกลงฉบับนี้จัดทำขึ้นเพื่อกำหนดแนวทางในการใช้บริการระหว่างผู้ใช้บริการ (“ผู้ใช้”) และผู้ให้บริการ (“บริษัท ดีเพย์อิท”) โดยผู้ใช้ตกลงที่จะปฏิบัติตามเงื่อนไขและข้อกำหนดที่ระบุไว้ในเอกสารนี้อย่างเคร่งครัด

1. ข้อกำหนดทั่วไป

1.1 วัตถุประสงค์: ดีเพย์อิท (Depayit) ทำหน้าที่เป็น "ตัวกลางอำนวยความสะดวก" ในการรับและจ่ายเงินสำหรับธุรกรรมซื้อขายสินค้าออนไลน์มือสอง โดยมิได้มีสถานะเป็นธนาคารหรือสถาบันการเงิน เราทำหน้าที่ดูแลธุรกรรมผ่านบัญชี Escrow จนกว่าเงื่อนไขการซื้อขายจะสมบูรณ์

1.2 การยอมรับข้อตกลง: การใช้งานแพลตฟอร์มถือว่าผู้ใช้ยอมรับข้อตกลงและนโยบายทั้งหมดที่เกี่ยวข้อง

2. การใช้บริการและขอบเขตความรับผิดชอบ

2.2 กระบวนการทำธุรกรรม (Escrow Model)

  • การสร้างธุรกรรม: ผู้ซื้อหรือผู้ขายกรอกรายละเอียดสินค้าและราคา ระบบจะสร้าง Link สำหรับชำระเงิน (QR Code) และ Link สำหรับติดตามสถานะ
  • การชำระเงิน: เงินของผู้ซื้อจะถูกพักไว้ในบัญชี Escrow และระบบจะแจ้งเตือนเมื่อเงินเข้าแล้ว
  • การจัดส่ง: ผู้ขายจัดส่งสินค้าและแจ้งเลข Tracking Number เพื่อยืนยันในระบบ
  • การตรวจสอบและรับเงิน: เมื่อผู้ซื้อได้รับของและกดยืนยัน (Confirm) หรือเมื่อครบกำหนดเวลา (Timeout 48 ชม.) โดยไม่มีการแจ้งปัญหา ระบบจะโอนเงินให้ผู้ขายอัตโนมัติ (Auto-Release)

2.3 การระงับข้อพิพาท (Dispute)

หากสินค้ามีปัญหา ผู้ซื้อต้องกด "เปิดข้อพิพาท" ภายใน 48 ชั่วโมง นับจากสินค้าถึงปลายทาง (อ้างอิงตาม Tracking Number)

  • ต้องแนบหลักฐานภาพถ่าย หมายเลขธุรกรรม (TXID) และระบุเหตุผลชัดเจน
  • หากไม่ดำเนินการในเวลาที่กำหนด เงินจะถูกโอนให้ผู้ขายตามเงื่อนไข Auto-Release
  • การตัดสินของบริษัทฯ ถือเป็นที่สิ้นสุด

2.4 นโยบายการคืนเงินและการส่งสินค้าคืน

  • การยกเลิก: ทำได้ก่อนผู้ขายจัดส่งสินค้า (ได้รับเงินคืนเต็มจำนวน หักค่าธรรมเนียมตามจริง)
  • การส่งคืนสินค้า: หากผู้ซื้อชนะข้อพิพาท ต้องส่งสินค้าคืนภายใน 7 วันทำการ พร้อมแจ้งเลข Tracking ในระบบ
  • บทลงโทษ: ผู้ซื้อที่มีประวัติส่งคืนไม่ตรงหรือทุจริต จะถูกบันทึกในระบบ Blacklist และอาจถูกระงับการใช้งาน[span_15

3. ข้อจำกัดความรับผิด

บริษัทฯ ไม่รับผิดชอบต่อคุณภาพสินค้า หรือการผิดสัญญาของคู่ค้า และขอยืนยันสถานะว่า "ไม่ใช่ธนาคาร" หรือผู้ให้บริการรับฝากเงินตามกฎหมาย ผู้ใช้บริการยอมรับความเสี่ยงที่เกี่ยวข้องตามสมควร

4. นโยบายความปลอดภัยและยืนยันตัวตน (KYC/AML)

เพื่อให้สอดคล้องกับกฎหมายป้องกันการฟอกเงิน (AML) เรากำหนดระดับการยืนยันตัวตนดังนี้

  • Tier 1 วงเงิน < 30,000 บาท/ครั้ง: ยืนยันเบอร์โทรศัพท์และอีเมล
  • Tier 2 วงเงิน < 100,000 บาท/ครั้ง: ยืนยันตัวตนด้วยบัตรประชาชนและรูปถ่าย (Selfie)
  • Tier 3 วงเงิน > 100,000 บาท/ครั้ง: ต้องทำ e-KYC ผ่าน NDID หรือ API ธนาคาร

5. นโยบายข้อมูลส่วนบุคคล (PDPA)

บริษัทเก็บรวบรวมข้อมูลเท่าที่จำเป็น (เช่น ชื่อ, เบอร์โทร, เลขบัญชี) เพื่อการทำธุรกรรมและการยืนยันตัวตนเท่านั้น โดยมีมาตรการรักษาความปลอดภัยข้อมูลตามมาตรฐาน

ติดต่อเจ้าหน้าที่คุ้มครองข้อมูล (DPO)
